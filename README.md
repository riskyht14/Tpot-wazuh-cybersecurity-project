# Cloud-Based Honeypot and SIEM Cybersecurity Monitoring System

## Project Overview
This project demonstrates the design and deployment of a cloud-based cybersecurity monitoring
environment using T-Pot honeypots integrated with Wazuh SIEM.

The system captures real-world attack attempts, processes logs, and visualizes security events
through centralized dashboards.

## Architecture Components
- T-Pot Multi-Honeypot Platform
- Wazuh SIEM (Manager & Agent)
- Docker Containers
- Cloud Virtual Machines
- Network Security Controls

## Key Features
- Live attack simulation and capture
- Centralized log monitoring
- Security alert generation
- Threat modeling using STRIDE
- Incident response workflow implementation

## Technologies Used
- Linux
- Docker
- T-Pot
- Wazuh
- Cloud Infrastructure

## Security Notice
All configuration files are sanitized.
No real IP addresses, credentials, or sensitive data are included.
